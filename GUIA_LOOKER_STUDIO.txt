1. CONFIGURAÇÃO INICIAL
================================================================================

1.1 Conectar a Base de Dados
-----------------------------
1. Acesse: https://lookerstudio.google.com
2. Clique em "Criar" → "Relatório"
3. Clique em "Criar fonte de dados"
4. Escolha "Google Planilhas"
5. Selecione a planilha "Backup_Looker_Studio"
6. Selecione a aba "Dados"
7. Clique em "Conectar"
8. Clique em "Adicionar ao relatório"

1.2 Configurar Tema
-------------------
1. Clique em "Tema e layout" (ícone de paleta)
2. Configure:
   - Cor primária: #009688
   - Cor secundária: #455a64
   - Fonte: Roboto

1.3 Configurar Página
---------------------
1. Arquivo → Configurações do relatório
2. Tamanho da página: Personalizado 1200 x 1800
   (ou escolha "Responsivo")

================================================================================
2. CAMPOS CALCULADOS
================================================================================

Antes de criar gráficos, crie os campos calculados.

2.1 Acessar Editor de Campos
-----------------------------
1. Clique em "Recurso" → "Gerenciar fontes de dados adicionadas"
2. Clique no nome da sua fonte de dados
3. Clique em "EDITAR" (canto superior direito)
4. Clique em "ADICIONAR UM CAMPO" (botão azul)

2.2 Criar Cada Campo
--------------------
Crie os campos abaixo, um por vez. Após cada um, clique em "SALVAR".

Campo 1:
Nome: Tempo Minutos Num
Fórmula: CAST(Tempo_Resposta_Minutos AS NUMBER)

Campo 2:
Nome: Tempo Horas Num
Fórmula: CAST(REPLACE(Tempo_Resposta_Horas, ",", ".") AS NUMBER)

Campo 3:
Nome: Tempo Dias Num
Fórmula: CAST(REPLACE(Tempo_Resposta_Dias, ",", ".") AS NUMBER)

Campo 4:
Nome: Dentro SLA Flag
Fórmula: CASE WHEN Dentro_SLA = "SIM" THEN 1 ELSE 0 END

Campo 5:
Nome: Fora SLA Flag
Fórmula: CASE WHEN Dentro_SLA = "NÃO" THEN 1 ELSE 0 END

Campo 6:
Nome: Com Diferenca Valor Flag
Fórmula: CASE WHEN Houve_Diferenca_Valor = "SIM" THEN 1 ELSE 0 END

Campo 7:
Nome: SLA No Prazo Flag
Fórmula: CASE WHEN Status_SLA = "OK" THEN 1 ELSE 0 END

Campo 8:
Nome: SLA Alerta Flag
Fórmula: CASE WHEN Status_SLA = "ALERTA" THEN 1 ELSE 0 END

2.3 Finalizar
-------------
Clique em "CONCLUÍDO" para voltar ao relatório.
